# 古琴助手小程序说明文档

## 项目简介
本项目是一个面向古琴学习者的微信小程序，定位为“随手可用的古琴练习辅助工具箱”。当前版本围绕日常练习场景，提供调音、节拍、录音、减字谱检索、琴馆检索、知识查阅、打卡与回顾等功能。

## 产品定位
- 面向古琴初学者与持续练习者。
- 强调本地可用、轻量打开、工具直达。
- 在练习前（调音 / 节拍）、练习中（录音 / 查阅）、练习后（打卡 / 回顾）提供连续辅助。

## 主要功能模块
### 1. 调音器
目录：`pages/tools/tuner/`。

当前实现包含以下能力：
1. 支持多组调式预设并按当前弦位显示目标频率。
2. 点击“开始监听”后申请录音权限并启动实时音频帧监听。
3. 内置多组 recorder profile 自动降级，尽量兼容不同机型与基础库差异。
4. 成功工作的 recorder profile 会写入本地缓存，后续优先复用，提高“即开即用”成功率。
5. 即使还未形成稳定音高，也会尽早显示当前声音输入的估计频率、音量、稳定度和帧间隔。
6. 在检测到稳定音高后，会计算与当前目标弦位的偏差，并更新中间可视化与状态提示。

### 2. 节拍器
目录：`pages/tools/metronome/`。

功能说明：
- 支持 BPM 滑动调整与快捷增减。
- 支持 2 / 3 / 4 / 6 拍和散板模式。
- 提供 Tap Tempo。
- 通过声音与振动同步提示节拍。

### 3. 录音器
目录：`pages/tools/recorder/` 与 `pages/tools/recorder-detail/`。

功能说明：
- 支持练习录音、暂停、继续、保存。
- 本地保存录音文件与元信息。
- 支持录音详情查看、重命名、导出与分享。

### 4. 琴馆检索
目录：`pages/tools/qinshe/`、`pages/tools/qinshe-city/`、`pages/tools/qinshe-venue/`。

功能说明：
- 按省份查看收录琴馆数量。
- 按城市查看收录琴舍数量。
- 支持名称 / 城市 / 省份检索。
- 数据中补充了区县维度，便于后续扩展筛选能力。

### 5. 工具分类
- 练习工具：`pages/tools/tuner/`、`pages/tools/metronome/`、`pages/tools/recorder/`、`pages/tools/ywt/`。
- 古琴知识：`pages/tools/notation/`、`pages/tools/knowledge/`。
- 发现更多：`pages/tools/qinshe/`、`pages/tools/qinshi/`、`pages/tools/qinpu/`。
- 打卡、记录、回顾、个人页：`pages/checkin/`（含 records / review）、`pages/profile/`（含 about）。

## 调音器重点说明
### 监听逻辑
调音器当前监听链路如下：
1. 请求隐私授权与录音权限。
2. 启动 recorder profile。
3. 若启动失败、无帧返回或 profile 不兼容，则自动切换下一档参数。
4. 收到音频帧后先转为 `Int16Array`，兼容 PCM / WAV 数据。
5. 通过 YIN 检测与零交叉估算结合，尽早反馈声音输入。
6. 对稳定音高计算目标差值，并映射到可视化滑条。

### 已知调试建议
- 真机测试优先，尤其是 Android 设备。
- 若某设备首次进入调音器较慢，可多观察一次成功后的后续表现；成功 profile 会被缓存。
- 外部环境噪声过强时，可能会出现“已检测到声音，但音高尚未稳定”的提示。

## 命名与品牌
项目当前统一对外命名为 **古琴助手**。若后续修改应用名称、页面标题、分享文案等，请保持一致。

## 开发与调试建议
- 使用微信开发者工具打开项目根目录进行调试。
- 如遇基础库兼容性提示，建议切换稳定版本进行验证。
- 录音相关功能依赖 `scope.record` 权限，请在真机或已授权环境中测试。
- 涉及小程序缓存的数据（如调音器成功 profile、录音列表）时，可在开发者工具中清缓存后重新验证冷启动行为。

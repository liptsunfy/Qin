# 古琴习练助手小程序说明文档

## 项目简介
本项目是一个面向古琴学习者的微信小程序，提供调音器等常用工具，帮助用户快速进入习练状态。

## 主要功能
- 调音器：提供调式预设、音高偏差提示与自动推进功能。
- 隐私协议：启动时自动检测隐私协议授权状态并提示用户处理。

## 调音器说明
调音器页面位于 `pages/tools/tuner/`，当前实现包含以下逻辑：
1. 启动时加载调式预设并应用默认预设。
2. 切换弦位或调式时会刷新当前调音反馈。
3. 点击“开始监听”后会申请录音权限并启动录音帧回调。
4. 录音帧回调会执行音高检测、偏差计算并更新 UI 状态。
5. 调式预设以 C 调为基准（例如正调：五六一二三五六 → G2/A2/C3/D3/E3/G3/A3）。

## 开发与调试建议
- 使用微信开发者工具打开项目根目录进行调试。
- 如遇基础库兼容性提示，建议在开发者工具中切换至稳定版本进行验证。
- 录音相关功能依赖 `scope.record` 权限，请在真机或已授权环境中测试。

## 目录结构（节选）
- `app.js`：小程序生命周期与全局错误处理。
- `pages/tools/tuner/`：调音器页面实现。
- `docs/`：项目说明文档。
- `CHANGELOG.md`：版本更新日志。
